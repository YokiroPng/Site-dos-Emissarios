# **App Name**: Emissários

## Core Features:

- Emissary Profiles: Display profiles for each VTuber with their icon, name, description, and social media links.
- Interactive Menu: Fixed top navigation bar with links to 'Início', 'Emissários', 'Gerações', 'Horários', and 'Contato' with subtle hover effects.
- Schedule Table: A clean and simple schedule table displaying VTuber, Time, Platform, and Link with optional thin gold line separators.
- Footer Information: A footer section with a mission statement and contact information.

## Style Guidelines:

- Primary color: Gold (#FFD700) for titles and details to create a luxurious and mystical feel.
- Background color: Dark background (#121212) for a minimalist and mysterious theme.
- Accent color: White (#FFFFFF) for important icons and texts, providing clear contrast.
- Font: The existing serif font, as used on the reference website.
- Fluid layout with no frames or centered boxes for a clean and modern design. Emissary profiles aligned to the left.
- Subtle hover animations on icons and menu items, with a soft gold glow effect.
- Use white icons for social media links, with slight gold sheen when clicked.
